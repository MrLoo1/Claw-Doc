# 第6章 定时任务

OpenClaw 支持定时任务，可以设置提醒和周期性操作。

## 创建提醒

### 通过对话

```
提醒我明天早上9点开会
```

```
每周一早上10点提醒我写周报
```

### 查看任务

```
查看我的提醒
```

### 取消任务

```
取消明天的提醒
```

## Cron 表达式

支持标准 cron 格式：

```
┌───────────── 分钟 (0-59)
│ ┌───────────── 小时 (0-23)
│ │ ┌───────────── 日 (1-31)
│ │ │ ┌───────────── 月 (1-12)
│ │ │ │ ┌───────────── 星期 (0-6)
│ │ │ │ │
* * * * *
```

### 常用示例

| 表达式 | 含义 |
|--------|------|
| `0 9 * * *` | 每天早上9点 |
| `0 9 * * 1` | 每周一早上9点 |
| `*/30 * * * *` | 每30分钟 |
| `0 18 * * 1-5` | 工作日下午6点 |

## 配置

```yaml
cron:
  enabled: true
  timezone: "Asia/Shanghai"
```
