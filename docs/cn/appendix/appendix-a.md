# 附录 A：命令速查表

OpenClaw CLI 命令快速参考。

## 基础命令

| 命令 | 说明 |
|------|------|
| `openclaw --version` | 查看版本 |
| `openclaw --help` | 查看帮助 |
| `openclaw init` | 初始化工作区 |
| `openclaw start` | 启动服务 |
| `openclaw stop` | 停止服务 |
| `openclaw status` | 查看状态 |

## 配置命令

| 命令 | 说明 |
|------|------|
| `openclaw config` | 查看配置 |
| `openclaw config set <key> <value>` | 设置配置 |
| `openclaw config get <key>` | 获取配置项 |

## 渠道命令

| 命令 | 说明 |
|------|------|
| `openclaw channel list` | 列出渠道 |
| `openclaw channel enable <name>` | 启用渠道 |
| `openclaw channel disable <name>` | 禁用渠道 |

## 会话命令

| 命令 | 说明 |
|------|------|
| `openclaw chat` | 开始对话 |
| `openclaw sessions` | 查看会话列表 |
| `openclaw clear` | 清除历史 |

## 技能命令

| 命令 | 说明 |
|------|------|
| `openclaw skill list` | 列出技能 |
| `openclaw skill install <name>` | 安装技能 |
| `openclaw skill enable <name>` | 启用技能 |

## Gateway 命令

| 命令 | 说明 |
|------|------|
| `openclaw gateway start` | 启动网关 |
| `openclaw gateway stop` | 停止网关 |
| `openclaw gateway restart` | 重启网关 |
| `openclaw gateway status` | 网关状态 |
