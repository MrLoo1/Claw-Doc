# 应用场景

OpenClaw 可以应用于多种场景，帮助你提升效率和自动化工作流程。

## 🌅 个人效率

| 场景 | 说明 |
|------|------|
| 早间简报 | 天气 + 日程 + 待办事项自动推送 |
| 邮件自动分类 | 自动分类与摘要邮件内容 |
| 智能日程管理 | 日历集成，智能提醒和安排 |

### 示例配置

```yaml
# 早间简报定时任务
cron:
  - schedule: "0 8 * * *"
    task: morning-briefing
    channels: [telegram, qqbot]
```

## 💻 编程开发

| 场景 | 说明 |
|------|------|
| 代码生成 | 根据需求自动生成代码片段 |
| 代码审查 | 自动检查代码质量和规范 |
| 自动化测试 | 生成测试用例并执行 |
| 文档生成 | 自动生成 API 文档和注释 |

### 技能推荐

- `code-review` - 代码审查技能
- `test-generator` - 测试生成技能
- `doc-generator` - 文档生成技能
