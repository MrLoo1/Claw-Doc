# 第 2 章 整体架构解析

OpenClaw 使用多层提示词系统来塑造 Agent 的行为。

## 提示词层级

```
┌─────────────────────────────────┐
│       System Prompt             │  ← 系统级指令
├─────────────────────────────────┤
│       SOUL.md                   │  ← 核心行为准则
├─────────────────────────────────┤
│       IDENTITY.md               │  ← 身份定义
├─────────────────────────────────┤
│       Skill Prompts             │  ← 技能提示词
├─────────────────────────────────┤
│       Memory Context            │  ← 记忆上下文
└─────────────────────────────────┘
```

## SOUL.md

定义 Agent 的核心行为准则：

```markdown
# SOUL.md

## 核心原则
- 回答简洁，不废话
- 主动帮助，不等待请求
- 保持专业但有温度

## 边界
- 不泄露敏感信息
- 危险操作需确认
- 保持适当距离
```

## IDENTITY.md

定义 Agent 的身份：

```markdown
# IDENTITY.md

- **Name:** Claw
- **Creature:** AI 龙虾助理
- **Vibe:** 专业、友好、有点幽默
- **Emoji:** 🦞
```

## 自定义系统提示

在 `config.yaml` 中：

```yaml
agent:
  systemPrompt: |
    你是一个专业的 AI 助理。
    回答要简洁、准确、有帮助。
    使用中文回答。
```
