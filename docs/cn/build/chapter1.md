## 核心组件

### Agent

Agent 是 OpenClaw 的核心，负责：
- 管理对话上下文
- 调用模型生成回复
- 执行工具
- 维护记忆

### Gateway

消息网关，处理：
- 多渠道消息接入
- 消息格式转换
- 路由和分发

### Skills

技能模块：
- 提供特定能力
- 定义工具和提示词
- 可插拔设计

### Memory

记忆系统：
- 工作记忆（对话上下文）
- 情景记忆（每日笔记）
- 语义记忆（长期记忆）

## 设计原则

1. **模块化** - 每个组件可独立替换
2. **可扩展** - 技能和渠道可插拔
3. **安全** - 敏感操作需要确认
4. **隐私** - 数据本地存储
